ContentGraph – Quick Start

Focus on four pieces: Contentful Sidebar, Publish Webhook, Slack /cg, and Analytics Mix.

1) Contentful App (Sidebar)

Location URL: /app/sidebar.html
  • Assign to your target content type(s).
  • Ensure fields for title/body exist (auto-detected).
  • Open an entry and click Deep Crawl & Classify, then Save Draft or Publish.
Open classification preview (outside Contentful)
Fallback (if sidebar button missing): Open Standalone Classify Page

2) Contentful Webhook (Auto-classify)

Webhook URL: /webhooks/contentful
  • Trigger: On Entry publish for your content types.
  • Queues classification processing and writes approved results back to Contentful.

3) Slack Slash Command

Command: /cg → Request URL: /slack/cg
persona=
funnel=
industry=
topic=
limit=
public=true
Example: /cg persona=decision-makers funnel=consideration industry=finance limit=3

4) Analytics Quick View

Open the funnel mix chart: /app/mix.html
Or JSON: /analytics/content-mix

Salesforce (Q&A)

  • Phase 1: Use /cg in sales channels (public=true to post in-channel).
  • Phase 2: Lightning component calls /graph/search using a Named Credential.
